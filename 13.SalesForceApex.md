# Salesforce Application using Apex – Selection Sort (Greedy Algorithm)

## Objective
To create an application on Salesforce using Apex programming language and implement the Selection Sort algorithm.

---

## Requirements
- Salesforce Developer Account (https://developer.salesforce.com)
- Basic knowledge of Apex programming
- Web browser

---

## Introduction
Salesforce is a cloud-based CRM platform that allows developers to build applications using Apex, a Java-like programming language. Apex is used to write business logic and execute operations on Salesforce data.

Selection Sort is a simple sorting algorithm based on the greedy approach. It repeatedly selects the smallest element from the unsorted portion of the list and places it at the correct position.

## Apex Code for Selection Sort

```apex
public class SelectionSortDemo {

    // method to perform selection sort
    public static List<Integer> selectionSort(List<Integer> arr) {
        Integer n = arr.size();

        for (Integer i = 0; i < n; i++) {
            Integer minIndex = i;

            // find minimum element
            for (Integer j = i + 1; j < n; j++) {
                if (arr[j] < arr[minIndex]) {
                    minIndex = j;
                }
            }

            // swap elements
            Integer temp = arr[i];
            arr[i] = arr[minIndex];
            arr[minIndex] = temp;
        }

        return arr;
    }

    // test method
    public static void testSort() {
        List<Integer> numbers = new List<Integer>{64, 25, 12, 22, 11};
        List<Integer> sortedList = selectionSort(numbers);

        System.debug('Sorted List: ' + sortedList);
    }
}
```
