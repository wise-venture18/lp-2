# Create and Launch a Virtual Machine using Amazon EC2

## Objective
To create and launch a virtual machine instance using Amazon EC2.

---

## Requirements
- AWS Account
- Internet Connection
- Basic knowledge of cloud computing

---

## Introduction
Amazon EC2 (Elastic Compute Cloud) is a service provided by AWS that allows users to create and manage virtual machines (instances) in the cloud. It provides scalable computing capacity and allows users to run applications without managing physical hardware.

---

## Steps to Launch EC2 Instance

### 1. Sign in to AWS
- Go to https://aws.amazon.com  
- Click on **Sign In to Console**  
- Enter your credentials  

---

### 2. Open EC2 Dashboard
- In the AWS search bar, type **EC2**  
- Click on **EC2 Dashboard**  

---

### 3. Launch Instance
- Click on **Launch Instance**  
- Enter instance name (e.g., `MyServer`)  
<img width="626" height="346" alt="image" src="https://github.com/user-attachments/assets/69e28ec3-c4ec-48ca-8f0f-6f0c1975be07" />

---

### 4. Choose AMI (Operating System)
Select an Amazon Machine Image (AMI):
- Amazon Linux  
- Ubuntu  
- Windows Server  

---

### 5. Choose Instance Type
- Select **t2.micro** (Free Tier eligible)  

---

### 6. Create Key Pair
- Click **Create new key pair**  
- Enter name (e.g., `mykey`)  
- Download the `.pem` file  
- Store it securely  

---

### 7. Configure Network Settings
- Allow:
  - SSH (Port 22) for Linux  
  - RDP (Port 3389) for Windows  
- Default VPC can be used  

---

### 8. Configure Storage
- Default: 8 GB  
- Modify if required  

---

### 9. Launch Instance
- Click **Launch Instance**  
- Wait until instance state becomes **Running**  

---

### 10. Connect to Instance

#### For Linux (SSH)
```bash
ssh -i mykey.pem ec2-user@your-public-ip
```
#### For Windows (RDP)
Download RDP file
Use password generated from key pair
---
### 11. Verify Instance
Instance State: Running
Status Checks: Passed
Result

Successfully created and launched a virtual machine using Amazon EC2.
